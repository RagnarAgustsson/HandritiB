**backend/app/config.py:**
- Use pydantic_settings BaseSettings

- Parse CORS_ORIGINS as comma-separated string into list via a property/validator
- class Config: env_file = ".env"

**backend/app/main.py:**
- Create FastAPI app with title="Handriti Backend", version from config
- Add CORS middleware using origins from config (per CONTEXT.md: strict production origins, localhost via env)
- Include health router
- Store app start time for uptime calculation
- Lifespan context manager for startup/shutdown (store start_time in app.state)

**backend/app/routers/health.py:**
- `GET /health` endpoint returning JSON:
  - `status`: "ok" or "degraded"
  - `version`: from config
  - `uptime_seconds`: float, calculated from app start time
  - `timestamp`: ISO 8601 UTC
  - `database`: "connected" or "unreachable" (test with a simple `SELECT 1` using sqlalchemy, catch exceptions)
- Use a sync or async approach for the DB check. Since we have asyncpg, use SQLAlchemy async engine with `create_async_engine`. But for simplicity in the health check, a sync check with psycopg2 is also fine. Choose the simpler approach.
- If DB check fails, still return 200 but with `status: "degraded"` and `database: "unreachable"` — the health check should not fail just because DB is temporarily down (Railway would restart the service).

<task type="auto">
  <name>Task 2: Create Dockerfile for Railway deployment</name>
  <files>backend/Dockerfile</files>
  <action>
Create a multi-stage Dockerfile optimized for Railway:

```dockerfile
FROM python:3.12-slim AS builder
```

**Stage 1 (builder):**
1. Install uv via pip or copy from official image (`COPY --from=ghcr.io/astral-sh/uv:latest /uv /usr/local/bin/uv`)
2. Set workdir to /app
3. Copy pyproject.toml and uv.lock first (layer caching)
4. Run `uv sync --frozen --no-dev` to install production deps only
5. This creates .venv in /app

**Stage 2 (runtime):**
1. FROM python:3.12-slim
2. Copy .venv from builder stage
3. Copy app/ source code
4. Set PATH to include .venv/bin
5. Expose port 8000
6. Health check: `HEALTHCHECK CMD python -c "import urllib.request; urllib.request.urlopen('http://localhost:8000/health')" || exit 1`
7. CMD: `uvicorn app.main:app --host 0.0.0.0 --port 8000`

Use `$PORT` env var from Railway if available: `CMD uvicorn app.main:app --host 0.0.0.0 --port ${PORT:-8000}`

Keep the image minimal — no dev tools, no build artifacts beyond .venv.
