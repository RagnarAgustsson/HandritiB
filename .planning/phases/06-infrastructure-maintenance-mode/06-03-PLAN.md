<!-- From research/05-vercel-railway-coexistence.md -->
Clerk JWT verification pattern:
```python
from fastapi_clerk_auth import ClerkConfig, ClerkHTTPBearer
clerk_config = ClerkConfig(jwks_url="https://.../.well-known/jwks.json")
clerk_auth_guard = ClerkHTTPBearer(config=clerk_config)

@app.get("/protected")
async def protected(credentials=Depends(clerk_auth_guard)):
    return credentials  # decoded JWT payload
```

SQLAlchemy connection pattern:
- Use pooled connection string (hostname with -pooler suffix)
- pool_pre_ping=True, poolclass=NullPool (avoid double-pooling with PgBouncer)
